<h2>Multi-Tool Clinical Assistant MCP Server</h2>

<ul>
<li>Developed a Model Context Protocol (MCP) server enabling a multi-tool LLM agent for clinical decision support and hospital workflow automation.</li>
<li>Integrated medical APIs and structured SQL patient databases for real-time patient lookup, lab result interpretation, and risk scoring.</li>
<li>Designed modular tool-calling architecture to support scalable and extensible clinical reasoning workflows.</li>
</ul>
